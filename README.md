## Introduction

This portfolio is designed to provide an overview of my professional journey, including my projects, work experience, and educational background. It is built using HTML, CSS, and JavaScript, with a focus on responsive design and user experience.

## Features

- **Theme Toggle**: Switch between light and dark themes.
- **Responsive Navigation**: Hamburger menu for mobile devices.
- **Dynamic Sections**: Navigate through different sections without reloading the page.
- **Slideshow**: Interactive slideshows for project images.
- **Tooltips**: Additional information on hover for certain elements.

## Technologies Used

- HTML
- CSS
- JavaScript

## Setup

To run this project locally, follow these steps:

1. Clone the repository:
  ```bash
  git clone https://github.com/mohanad-80/portfolio.git
  ```
2. Navigate to the project directory:
  ```bash
  cd portfolio
  ```
3. Open `index.html` in your preferred web browser.

## Usage

- Click on the navigation links to switch between sections.
- Use the theme toggle button to switch between light and dark themes.
- On the projects page, view project details and navigate through slideshows.
